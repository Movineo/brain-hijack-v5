Core Principles

1. Trade Psychology, Not Assets
We don't predict prices. We detect when the herd's brain is being hijacked by emotion (FOMO, FUD, euphoria, panic) and position accordingly.
2. Contrarian at Extremes
When everyone is euphoric (extreme greed), we prepare to sell. When everyone has capitulated (extreme fear), we prepare to buy. The crowd is wrong at extremes.
3. Momentum in the Middle
Between extremes, we ride momentum with the trend. FOMO drives prices up, panic drives them down. Don't fight the trend until it exhausts.
4. Psychopathic Detachment
We observe human emotion without experiencing it. Their fear is our signal. Their greed is our exit. We are scientists studying the herd.

üìä Key Metrics

Primary Indicators

Hijack Force
A proprietary momentum indicator measuring the strength of the current brain hijack. Combines price velocity, volume, and order flow imbalance.
Force > 0.01 = Strong hijack in progress | Force > 0.005 = Moderate | Force < 0.002 = Weak
Fear & Greed Index
Market-wide sentiment indicator (0-100). Aggregates volatility, momentum, social sentiment, surveys, and derivatives data.
0-20 = Extreme Fear | 20-40 = Fear | 40-60 = Neutral | 60-80 = Greed | 80-100 = Extreme Greed
Sentiment Velocity
Rate of change in sentiment per hour. CRITICAL for early detection - catches moves BEFORE extremes hit.
Velocity > +5 = FOMO accelerating | Velocity < -5 = Panic accelerating
Hijack Strength
Overall confidence in the brain hijack signal (0-100%). Combines all indicators into a single actionable score.
60%+ with aligned signals = Trade candidate | 80%+ = High conviction

Supporting Indicators

Social Sentiment Score
Aggregated sentiment from Twitter/X, Reddit, and crypto social feeds. Ranges from -10 (extremely bearish) to +10 (extremely bullish).
Options Put/Call Ratio
Ratio of put options to call options. Low P/C (< 0.5) = FOMO, heavy call buying. High P/C (> 1.5) = Fear, heavy put buying/hedging.
On-Chain Health Score
Network fundamentals score (0-100). Considers active addresses, transaction volume, exchange flows, TVL. High score = strong fundamentals.
Whale Activity
Large holder transaction monitoring. Smart money accumulating/distributing can signal reversals before retail catches on.

üì° Signal Types (Triggers)

FOMO Signal
Detected when greed is rising - social sentiment bullish, F&G increasing, call options buying. Indicates momentum LONG opportunity.
FUD Signal
Detected when fear is rising - social sentiment bearish, F&G decreasing, put options buying. Indicates momentum SHORT or wait for capitulation.
MOMENTUM Signal
Detected when Hijack Force is strong - ride the wave regardless of sentiment. Pure price/volume momentum.
CONTRARIAN Signal
Detected at sentiment EXTREMES - go opposite the crowd. Buy extreme fear (F&G < 20), sell extreme greed (F&G > 80).
SMART_MONEY Signal
Detected from options flow and on-chain whale activity. Follow what large players are doing, not what retail is saying.
VELOCITY Signal
Detected from rate of sentiment change. Early warning system - catches moves before they peak.
PATTERN Signal
Detected when current conditions match a known herd behavior pattern from the database.

üîê Authentication

Brain Hijack uses API key authentication for protected endpoints.

Getting Your API Key

When the server starts, an admin API key is generated and logged to the console:

[AUTH] Admin API Key: a1b2c3d4...

You can also set a custom admin key via environment variable:

ADMIN_API_KEY=your_custom_key_here

Using Your API Key

Include the key in the X-API-Key header:

curl -H "X-API-Key: your_api_key_here" \
     https://your-domain.com/api/v1/sentiment/protected-endpoint

Permission Levels

Permission Access
read View data, get status, read signals
write Execute trades, modify settings
admin Full access, manage API keys, kill switch

Protected Endpoints

These endpoints require authentication:

  • POST /api/v1/sentiment/autotrader/start - Start the Brain Hijack Operator
  • POST /api/v1/sentiment/autotrader/stop - Stop the operator
  • POST /api/v1/sentiment/trade - Execute manual trades
  • POST /api/v1/sentiment/killswitch - Emergency stop all trading
